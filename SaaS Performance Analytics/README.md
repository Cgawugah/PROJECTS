# SaaS Performance Analytics Dashboard  
Power BI Project

This project presents a Power BI dashboard that analyzes key SaaS business metrics including customer lifetime value, acquisition cost, churn rate, active users, new users, marketing spend and recurring revenue.  
The dataset used for this report is intentionally small and simplified. This makes it ideal for rapid prototyping, demonstrating KPI logic and showcasing how core SaaS metrics can be modeled and visualized without requiring a large volume of data.

## Project Overview

The goal of this dashboard is to highlight essential SaaS performance indicators across a twelve month period.  
The report focuses on understanding how customer behavior, revenue and marketing investment change over time, and how these factors influence long term SaaS growth.

Despite the small dataset, the dashboard effectively demonstrates how SaaS businesses track retention, user activity, profitability and marketing effectiveness.

## Dashboard Features and Metrics

### Key Performance Indicators  
Average LTV  
Average CAC  
Average DAU to MAU ratio (stickiness)  

### User and Retention Metrics  
Active users by month  
Churn rate by month  
New users versus marketing spend  

### Revenue Metrics  
MRR by month  
Total revenue pattern over twelve months  

These metrics allow quick assessment of acquisition efficiency, retention strength and revenue stability.


## Key Insights

User activity and MRR show a gradual decline toward the end of the year  
Churn rate fluctuates but shows improvement in later months  
Marketing spend does not always correlate directly with new user growth  
LTV increases over time, indicating improving customer value  

These findings help illustrate how SaaS companies evaluate performance even within simple datasets.
